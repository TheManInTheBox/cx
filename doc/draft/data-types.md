Integral Types
--------------------------------------
byte - 1 byte unsigned integer
char - 1 byte signed char
wchar - 2 byte signed wide char
int - 4 byte signed integer

Fractional Types
--------------------------------------
float - 4 byte fractional value

Logical Types
--------------------------------------
bool - 1 bit values representing true or false
